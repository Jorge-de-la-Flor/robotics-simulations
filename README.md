# Robotics Simulations

Small simulation experiments exploring core robotics algorithms and system models.

This repository contains minimal implementations used to understand the behaviour of robotic systems such as mobile robots, sensor fusion pipelines, and feedback control loops.

The goal is to illustrate the **engineering principles behind robotics systems**, not to build production-ready frameworks.

---

## Topics explored

- mobile robot kinematics
- feedback control
- probabilistic localization
- occupancy grid mapping
- sensor fusion concepts

---

## Simulations

### Differential Drive Kinematics

Basic simulation of a differential drive robot illustrating how wheel velocities translate into robot motion.

### PID Motor Control

Simple closed-loop simulation showing how PID control stabilizes motor behaviour.

### Kalman Filter Localization

Minimal example demonstrating how Kalman filtering can estimate robot position from noisy sensor measurements.

### Occupancy Grid Simulation

Toy simulation illustrating probabilistic mapping using grid-based representations.

---

## Purpose

These simulations complement the repositories in the **Robotics & Systems Labs**, providing small executable models that demonstrate how robotics algorithms behave in practice.
